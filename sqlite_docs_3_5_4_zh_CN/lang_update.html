SQL As Understood By SQLite

UPDATE

sql-statement ::= UPDATE [ OR conflict-algorithm ] [database-name .] table-name
SET assignment [, assignment]*
[WHERE expr]
assignment ::= column-name = expr

UPDATE 语句用于修改数据表中所选行上列的值。 在UPDATE中，每一个赋值操作为左边是一个列名，中间是等号， 右边是一个表达式。表达式中可以使用其它列的值。 所有的表达式都会在赋值之前进行计算。 WHERE 子句用于限制更新哪些行。

可选的 conflict-clause 允许为UPDATE指定一个其它的冲突解决算法。详见 ON CONFLICT。
