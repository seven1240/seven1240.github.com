SQL As Understood By SQLite

DROP INDEX

sql-command ::= DROP INDEX [IF EXISTS] [database-name .] index-name

DROP INDEX 语句删除使用 CREATE INDEX 语句创建的索引。 索引名会从磁盘上完全删除，恢复索引的唯一办法是重新运行正确的 CREATE INDEX 命令。

DROP INDEX 语句在缺省模式下不会减少数据库文件的大小。 数据库中空闲的空间将会被后续的 INSERT 操作使用。 要移除数据库的自由空间，使用 VACUUM 命令。 如果一个数据库开启了 AUTOVACUUM 模式，那么 DROP INDEX 将自动释放空间。
